Dataset Information

How we collect data and what we do with it!

AMOS focuses on natural environments from webcams located around the world. The idea is to look at changes in these environment and analyze what could have brought up those changes. If a bike lane is implemented, we want to be able to quantify how many people use the bike lane versus the sidewalk.

Website

AMOS was designed using present and rising technologies. Flask and Python are the foundation of the frontend and backend, from capturing the data to hosting the website.

AMOS is designed using a relational database called postgresql. Future upgrades will be tied to a nosql database using MongoDB.

Images

Each webcam's image is captured every hour and the capture time is stored as the filename. The format is yyyymmdd_hhmmss.jpg (4 digits for year, and 2 digits each for month, day, hour minute, and second). We use a hash on each webcam that compares to the currently captured image to detect if the camera has been updated.
